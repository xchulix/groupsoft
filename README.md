# GroupSoft

GroupSoft es una plataforma tecnológica orientada a ofrecer soluciones profesionales de desarrollo web, software empresarial, ciberseguridad, administración de servidores Linux y servicios cloud.  
Este proyecto representa una web corporativa moderna, clara y enfocada en mostrar los servicios principales de la empresa.

🌐 **Demo online:** https://xchulix.github.io/groupsoft/

---

## 🚀 Características principales

- Diseño limpio, moderno y responsive  
- Estructura profesional orientada a empresa tecnológica  
- Secciones independientes para cada servicio  
- Navegación rápida y clara  
- Código organizado y fácil de mantener  
- Compatible con cualquier hosting estático (GitHub Pages, Netlify, etc.)

---

## 🧩 Tecnologías utilizadas

- **HTML5** — estructura y contenido  
- **CSS3** — estilos, diseño y animaciones  
- **JavaScript** — interacción y comportamiento  
- **GitHub Pages** — despliegue del sitio web

## 🎯 Objetivo del proyecto

El objetivo de este proyecto es presentar una web corporativa completa que muestre los servicios tecnológicos de GroupSoft, con un enfoque profesional y escalable.  
Además, sirve como ejemplo de:

- Diseño web moderno  
- Organización de contenido empresarial  
- Buenas prácticas en estructura de proyectos  
- Uso de GitHub Pages para despliegue
